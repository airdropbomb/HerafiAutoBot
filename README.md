# HerafiAutoBot-NTE
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/airdropbomb/HerafiAutoBot.git && cd HerafiAutoBot
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Edit .env file and input your private keys (one per line)

   ```bash
   nano .env
   ```

## Usage

- **Start the bot**:

  ```bash
  npm start
  ```
